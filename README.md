# 气象要素订正 (Weather Correction)

一个基于机器学习的气象数据订正系统，用于将网格再分析数据利用站点观测数据进行统计订正，提升气象要素的预报精度。


## 项目简介

气象数值预报模式输出的网格数据往往存在系统性偏差，本项目针对CARAS数据提供了一套完整的工作流程：

1. **数据导入**：导入站点观测数据和网格再分析数据
2. **数据处理**：数据清洗、特征工程、时间对齐
3. **模型训练**：使用机器学习算法训练订正模型
4. **数据订正**：应用模型对网格数据进行偏差订正
5. **效果评估**：多维度评估订正效果，支持批量站点评估

## 主要功能

### 1. 数据导入
- 自动读取站点观测数据文件
- 支持网格再分析数据（NetCDF格式）
- 数据自动入库，支持断点续传

### 2. 数据预览
- 可视化查看导入的站点数据
- 按站点、时间范围筛选数据
- 数据质量统计（缺失值、异常值等）

### 3. 数据处理
- 自动清洗异常值和缺失值
- 特征工程：时间特征、空间特征、地形特征
- 数据透视分析

### 4. 模型训练
- 支持 **XGBoost** 和 **LightGBM** 两种算法
- 自动超参数优化
- 支持早停策略防止过拟合
- 模型持久化保存

### 5. 数据订正
- 使用训练好的模型对网格数据进行订正
- 支持批量订正多个时间范围
- 输出订正结果和评估指标

### 6. 多站点评估
- 批量评估模型在多个站点的表现
- 生成详细的评估报告（JSON + Excel）
- 支持按季节筛选评估
- 多指标对比（CC、RMSE、MAE、MRE、MBE、R²）

### 7. 任务管理
- 异步任务执行，不阻塞API
- 实时进度查询
- 任务状态跟踪（PENDING、PROCESSING、COMPLETED、FAILED）
- 支持任务取消


## 支持的气象要素

| 要素名称 | 说明 | 订正方法 |
|---------|------|---------|
| 温度 | 2米气温 | 残差订正（模型预测 + 格点值） |
| 相对湿度 | 2米相对湿度 | 残差订正（模型预测 + 格点值） |
| 过去1小时降水量 | 1小时累计降水 | 残差订正（模型预测 + 格点值） |
| 2分钟平均风速 | 10米风速 | 直接订正（仅模型预测） |


### 评估指标

| 指标 | 全称 | 说明 | 越小越好/越大越好 |
|------|------|------|------------------|
| CC | 相关系数 | 衡量预报与观测的线性相关性 | ↑ 越大越好 |
| RMSE | 均方根误差 | 衡量预报误差的离散程度 | ↓ 越小越好 |
| MAE | 平均绝对误差 | 衡量预报误差的平均水平 | ↓ 越小越好 |
| MRE | 平均相对误差 | 相对误差的平均值 | ↓ 绝对值越小越好 |
| MBE | 平均偏差 | 系统性偏差的度量 | ↓ 绝对值越小越好 |
| R² | 决定系数 | 模型解释方差的比例 | ↑ 越大越好 |
